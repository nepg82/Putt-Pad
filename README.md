# Putt Pad — Mini Golf Scorecard

A small offline-friendly PWA for keeping score during a round of mini golf.

## What's inside
- `index.html`, `styles.css`, `app.js` — the app
- `manifest.json` — makes it installable ("Add to Home Screen")
- `sw.js` — service worker, caches the app shell so it works with no signal
- `icons/` — app icons

## Features
- Set course name, 9 or 18 holes, and par per hole (tap a par chip to cycle it)
- Add up to 6 players
- Score one hole at a time with big +/- steppers, or view the full scorecard grid
- Automatic Out/In/Total, relative-to-par per stroke, and a leader badge
- Everything saves to the device automatically — close the app mid-round and pick up where you left off
- Install to the home screen and it runs offline

## Why it needs to be hosted (quick note)
Browsers only treat a page as an installable, offline-capable PWA when it's served over **HTTPS** (or `localhost`) — not when opened directly from a file on disk. A couple of easy ways to get it there:

**Fastest — Netlify Drop (no account needed for a quick test)**
1. Go to https://app.netlify.com/drop
2. Drag this whole folder onto the page
3. Open the URL it gives you on your phone, then "Add to Home Screen"

**GitHub Pages (if you want a permanent link)**
1. Create a new GitHub repo and push these files to it
2. In the repo's Settings → Pages, set the source to your main branch
3. Visit the `https://<you>.github.io/<repo>/` URL it gives you

**Testing locally on your computer**
```
python3 -m http.server 8080
```
then open `http://localhost:8080` in your browser. Localhost counts as secure, so install/offline features work there too — just not on your phone unless it's the same machine.

## Customizing
- Colors and type live at the top of `styles.css` under `:root`
- Icon artwork is generated by `make_icons.py` if you want to tweak it (needs Pillow: `pip install pillow`)
